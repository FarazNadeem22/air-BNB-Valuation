# Portfolio Story

## Working Title

STR Investment Lab: API-driven underwriting and risk analysis for short-term rental acquisitions.

## Elevator Pitch

I built an end-to-end Python analytics product that ingests compliant real estate and STR market data, normalizes it into a reusable data model, underwrites acquisition scenarios, runs sensitivity analysis across financing and confidence levels, and supports optional machine learning models for revenue prediction.

## Skills Demonstrated

- Data engineering: API client boundaries, normalized domain models, raw-to-clean data flow
- Data analysis: return metrics, breakeven analysis, market comparison
- Data wrangling: source-specific adapters and canonical property/revenue objects
- Finance: mortgage math, cap rate, DSCR, cash-on-cash return, depreciation scenario framing
- Data science: ensemble revenue prediction as data volume grows
- Software engineering: package structure, tests, CLI, docs, typed interfaces, future deployment path

## Next Portfolio Milestones

1. Connect a licensed AirDNA endpoint and persist raw JSON.
2. Add SQLite tables for properties, comps, market metrics, and underwriting runs.
3. Export sensitivity matrices to CSV and Parquet.
4. Add a Streamlit dashboard for visual property comparison.
